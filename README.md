# TodoList
`https://basic-todolist-app.netlify.com/`
## Overview
Hi!,
Your time is precious, don't waste it. Use TodoList app to manage your tasks. It's simple, easy to use and free.

## Features
 - view a list of Todos
 - Mark a Todo as completed
 - Delete a Todo
 - Add a Todo
